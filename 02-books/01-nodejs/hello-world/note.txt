O método createServer do http cria um novo servidor HTTP
e o retorna.

É definido para o servidor escutar em uma porta e 
hostname.

Sempre em uma nova requisição é recebida, o evento de 
request é chamado, provendo dois objetos:

-> uma requisição (objeto to tipo http.IncomingMessage)
-> uma resposta (objeto do tipo http.ServerResponse)

Esses 2 objetos são essenciais para manusear a chamada
HTTP.

O primeiro provê os detalhes da requisição, com ele você
pode acessar os dados da request e as headers.

Neste caso, com:

res.statusCode = 200

nós definimos a propriedade statusCode como 200, para 
indicar uma resposta bem sucedida.

Nós definimos a header de Content-Type (tipo de conteúdo)

res.setHeader("Content-Type", "text/plain")

e nós fechamos a resposta, adicionando o conteúdo como um
argumento end()

resp.end("Hello World")
